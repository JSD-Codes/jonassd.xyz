Super Fighter Installation Guide

First, download the file here. If you have have not downloaded one of my games before, also download this and keep it in the same folder.

If IDLE is not already installed on your device, I reccomend reading this article.

Once you have IDLE installed, click on File > Open, then select super_fighter.py and click Run > Run Module. After that, enjoy the game!

If you are an advanced user and want to run it in a different IDE, go ahead! Just don't use the terminal or a web based interpreter. it does not work as well.
